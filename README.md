# CaseStudyMK / CaseStudyMotzetKunze

Dieses Repository dient als Grundlage für die Case Study im Rahmen der LV "Versionsverwaltung & Git".

## Inhalte
- `.gitignore` für Python
- `requirements.txt` für zukünftige Abhängigkeiten
- `main.py` (wird noch erstellt)
- Lokale virtuelle Umgebung `.venv` (wird nicht mit versioniert)

## Schritte für die Einrichtung
1. Repository klonen  
2. Virtuelle Umgebung erstellen: `python -m venv .venv`  
3. Aktivieren:  
   - Windows: `source .venv/Scripts/activate`
4. Abhängigkeiten installieren: `pip install -r requirements.txt`
5. `main.py` starten
